# TinyLink

TinyLink is a URL shortening web application that allows users to create short links, view click statistics, and manage their links efficiently. This project is built using Next.js and provides a user-friendly interface for managing URLs.

## Features

- **URL Shortening**: Users can input long URLs and generate short links.
- **Link Management**: Users can view, delete, and manage their shortened links.
- **Click Statistics**: Users can view detailed statistics for each link, including total clicks and last clicked time.
- **Dashboard**: A central dashboard to manage all links and access statistics.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/TinyLink.git
   cd TinyLink
   ```

2. Install the dependencies:
   ```
   npm install
   ```

3. Set up the environment variables:
   - Copy the `.env.example` file to `.env` and update the values as needed.

4. Run the development server:
   ```
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000` to access the application.

## Usage

- To shorten a URL, navigate to the link management page and fill out the form.
- View your shortened links on the dashboard, where you can also access statistics for each link.
- Click on a specific link to see detailed statistics, including a chart of click data.
